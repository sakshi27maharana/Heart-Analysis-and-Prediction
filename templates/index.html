Values in the dataset are defined as :

         1. caa : number of major vessels (0-3) colored by flourosopy.
         2. cp : Chest Pain type chest pain type
            Value 1: typical angina
            Value 2: atypical angina
            Value 3: non-anginal pain
            Value 4: asymptomatic
         3. restecg : resting electrocardiographic results
            Value 0: normal
            Value 1: having ST-T wave abnormality (T wave inversions and/or ST elevation or depression of > 0.05 mV)
            Value 2: showing probable or definite left ventricular hypertrophy by Estes' criteria
         4. slp : the slope of the peak exercise ST segment (1 = upsloping; 2 = flat; 3 = downsloping)
         5. thal : 3 = normal; 6 = fixed defect; 7 = reversable defect.
         6. thalachh : maximum heart rate achieved
     
Output of the prediction:

         0= less chance of heart attack
         1= more chance of heart attack
